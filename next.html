Question-2: What is the difference between innerHTML, innerText, and textContent ?

InnerHTML

Returns or sets the HTML content inside an element.

InnerText

Returns or sets the visible text inside an element, excluding hidden elements

TextContent

Returns or sets all text inside an element, including hidden text, without parsing HTML.

Question-4: What is event bubbling in the DOM?
Event bubbling is a mechanism in the DOM where an event starts from the target element and propagates upwards through its ancestor elements until it reaches the document object.
